# Product Overview

**Text to Prompt** is a browser extension that converts text from input fields into AI-optimized prompts. It works like LanguageTool by injecting a discreet lightbulb icon into text fields across the web.

## Core Features

- **Field Monitoring**: Automatically detects text fields (inputs, textareas, contenteditable) on any webpage
- **Smart Icon**: Displays a minimalist lightbulb icon in the bottom-right corner of focused fields  
- **AI Integration**: Supports OpenAI (GPT-5) and Google Gemini APIs
- **Three Conversion Modes**:
  - Text to English Prompt: Clear, concise English prompts
  - Text to JSON English Prompt: Structured JSON generation prompts
  - Text to JSON Prompt: JSON-focused prompts (any language)

## Target Platforms

- Chrome (Manifest V3)
- Firefox (Manifest V3)

## Privacy & Security

- All data stored locally using localStorage
- No external backend or tracking
- API keys stored securely in browser storage
- Open source and auditable
